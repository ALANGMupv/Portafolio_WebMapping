# Portafolio WebMapping

Portafolio digital desarrollado en la asignatura **Tecnologías de la Información Geográfica (TIG)**.
En este sitio se recopilan las prácticas realizadas durante el curso, aplicando herramientas de **GIS, análisis espacial, cartografía digital y visualización de datos geoespaciales**.

🌐 **Ver portafolio online:**
https://avazrem.upv.edu.es/TIGAlanAlex/index.html

---

## Descripción

Este portafolio académico reúne diferentes ejercicios prácticos relacionados con Sistemas de Información Geográfica (SIG), incluyendo:

* Análisis raster y vectorial
* Cartografía temática
* Teledetección
* Análisis espacial
* Desarrollo de **WebMaps interactivos**

Los resultados se presentan mediante distintos formatos como **PDF, imágenes, tablas y mapas web interactivos**.

---

## Tecnologías utilizadas

* HTML
* CSS
* JavaScript
* WebMapping
* QGIS

---

## Contenido

El portafolio incluye diversas prácticas del curso, como:

* Introducción a QGIS
* Mapas climáticos de temperatura y precipitación
* Análisis espacial de zonas válidas
* Índices de vegetación (SAVI y EVI)
* Análisis de severidad con datos raster
* Visualización de datos urbanos mediante WebMaps
