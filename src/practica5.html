PRÁCTICA 05

Severidad

Cálculo del índice dNBR y análisis de severidad mediante imágenes satélite

Formato final: WebMap
Software: QGIS
Ver resultados

Objetivo y Desarrollo de la Práctica

El objetivo de esta práctica es analizar la severidad de un incendio forestal mediante el tratamiento y análisis de imágenes satélite, utilizando índices espectrales en un Sistema de Información Geográfica. A partir de imágenes Sentinel-2 obtenidas antes y después del incendio, se calcula el índice NBR y posteriormente el índice dNBR, que permite estimar el grado de afectación de la vegetación.

El índice dNBR se reclasifica en distintos niveles de severidad, generando un raster de severidad del incendio que identifica las zonas más afectadas. Finalmente, los resultados se preparan para su visualización mediante un mapa web y para su análisis espacial posterior.

WebMap - Severidad

A continuación se muestra el mapa web interactivo con la capa resultante del análisis de severidad del incendio, obtenida a partir del índice dNBR. El mapa permite navegar mediante zoom y desplazamiento, así como consultar los valores de severidad mediante ventanas emergentes (pop-ups).
