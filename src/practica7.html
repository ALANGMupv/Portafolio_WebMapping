PRÁCTICA 07

Análisis de Rendimiento GNSS

Análisis numérico y representación gráfica de datos de rendimiento a partir de ficheros GNSS.

Formato final: Tabla/Gráficos
Herramienta: Python
Ver Resultados

Objetivo y Desarrollo de la Práctica

Esta práctica tiene como objetivo que el alumnado realice un análisis de rendimiento a partir de un fichero GNSS con waypoints en formato GPX. Se trata de una introducción al procesamiento de datos georreferenciados, donde el alumnado deberá aplicar diferentes librerías de Python para calcular y analizar parámetros espaciales y temporales a partir de coordenadas geográficas.

El resultado final incluirá tanto el análisis numérico como la representación gráfica e interactiva de los datos obtenidos.

Información técnica

  • Entrada: Fichero GNSS (waypoints en formato GPX).
  • Procesamiento: Uso de librerías Python para cálculo de parámetros.
  • Salida: Análisis numérico, gráficos y tablas de datos.
